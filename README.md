# YouTube Clone

## Purpose

The purpose of this repository is to store all the code for your web application. This also includes the history of all commits made and who made them.

# Build/Run Instructions

## Build Instructions
1. npm install
2. set up .env file

|       Variable        |      Value      |
|:-------------:|:---------------------:|
| DB_HOST  | hostname |
| DB_USER    | user             |
| DB_PASSWORD | password  |
| DB_NAME | name  |
| DB_PORT | port #  |
| PORT | port #  |
3. npm run builddb

## Run Instructions
1. npm start
